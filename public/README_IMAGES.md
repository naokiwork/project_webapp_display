# Project Images

Add your project screenshots to this `public` folder with the following filenames:

## Required Images

1. **Study Hour Calendar**
   - Filename: `study-hour-calendar.jpg` (or `.png`, `.webp`)
   - Screenshot of the calendar interface showing December 2025

2. **Category Search** (信条構成診断)
   - Filename: `category-search.jpg` (or `.png`, `.webp`)
   - Screenshot of the quiz interface with the first question

3. **Habit Grid**
   - Filename: `habit-grid.jpg` (or `.png`, `.webp`)
   - Screenshot of the habit tracking grid interface

## Image Guidelines

- **Format**: `.jpg`, `.jpeg`, `.png`, or `.webp`
- **Recommended size**: 800x600px or similar aspect ratio (16:9 or 4:3)
- **Optimization**: Compress images for web use to improve loading speed
- **Naming**: Use lowercase with hyphens (e.g., `study-hour-calendar.jpg`)

## How to Add Images

1. Save your screenshot/images with the exact filenames above
2. Place them in the `/public` folder
3. The images will automatically be available at `/study-hour-calendar.jpg`, `/category-search.jpg`, and `/habit-grid.jpg`

## Note

If images are not added, the project cards will display a placeholder with a GitHub icon.
