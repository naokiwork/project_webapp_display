# 画像ファイルの追加方法

以下の3つの画像ファイルを `public` フォルダに配置してください：

## 必要なファイル

1. **study-hour-calendar.jpg** (または .png, .webp)
   - Study Hour Calendarのスクリーンショット

2. **category-search.jpg** (または .png, .webp)
   - Category Search（信条構成診断）のスクリーンショット

3. **habit-grid.jpg** (または .png, .webp)
   - Habit Gridのスクリーンショット

## ファイルを配置する方法

### 方法1: Finderを使用（Macの場合）

1. Finderを開く
2. 以下のパスに移動：
   ```
   /Users/naokionda/Git-Hub_repository/project_webapp_display/public
   ```
3. 画像ファイルをこのフォルダにドラッグ＆ドロップ
4. ファイル名を上記の名前に変更

### 方法2: ターミナルを使用

1. 画像ファイルがデスクトップにある場合：
   ```bash
   cd ~/Desktop
   cp study-hour-calendar.jpg /Users/naokionda/Git-Hub_repository/project_webapp_display/public/
   cp category-search.jpg /Users/naokionda/Git-Hub_repository/project_webapp_display/public/
   cp habit-grid.jpg /Users/naokionda/Git-Hub_repository/project_webapp_display/public/
   ```

2. または、画像ファイルがある場所から直接コピー：
   ```bash
   cp [画像ファイルのパス]/*.jpg /Users/naokionda/Git-Hub_repository/project_webapp_display/public/
   ```

## 確認

ファイルが正しく配置されたか確認：
```bash
ls -la /Users/naokionda/Git-Hub_repository/project_webapp_display/public/
```

以下のファイルが表示されればOK：
- study-hour-calendar.jpg
- category-search.jpg
- habit-grid.jpg

## 画像の推奨設定

- **形式**: .jpg, .png, .webp
- **推奨サイズ**: 800x600px または類似のアスペクト比
- **ファイルサイズ**: 200KB以下を推奨（読み込み速度のため）
